# Markdown Previewer  
This is a **Markdown Previewer**. Write your Markdown here to see the magic!  

## Features:  
- Live preview of your Markdown.  
- Export as PDF or Word document.  
- Easy to use!

### Example Syntax:  
1. **Bold**: `**text**`  
2. *Italic*: `*text*`  
3. [Link](https://example.com)  
4. Inline code: `` `code` ``
